# eye dropper 

tab manager chrome extension ^^ 

## description 

this extension lets you pick colors from any website, giving you the hex code and rgba value. it's the first chrome extension i've coded! 

## execution 

1. clone this repository 
2. load this file in your browser as an **upacked extension** through extensions developer mode 
3. click the extension icon in the toobar and select the "tab manager" icon. a popup should appear, starting up the extension
